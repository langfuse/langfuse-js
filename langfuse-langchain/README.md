This is the JS/TS client for the Langfuse Langchain integration.

## Environments

**Supported**

- Node.js >=20
- Web
- Edge: Vercel, Cloudflare Workers, etc.

## Installation

```bash
npm i langfuse-langchain
# or
yarn add langfuse-langchain
# or
pnpm i langfuse-langchain
```
