# Langfuse JS/TS

This is the main JS/TS client for Langfuse.

**Supported environments**

- Node.js >=18
- Web
- Edge: Vercel, Cloudflare Workers, etc.

**Node.js <18**

Use `langfuse-node` instead as it does not use `fetch` and other Web APIs

## Installation

```bash
npm i langfuse
# or
yarn add langfuse
# or
pnpm i langfuse
```
